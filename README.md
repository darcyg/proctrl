# proctrl
Linux c++ library for controlling and  monitoring child processes.
# Usage
## Requirement
|Libraries| Versions|
|-------- |--------:|
|Boost    | >= 1.57 |
|g++      | >= 4.9  |
|cmake    | >= 2.8  |
## Instructions
```bash
mkdir build
cd build
cmake ..

make

make test
```
